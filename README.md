# Chinese Sports Notes

## 项目简介

本仓库用于归档与发布多个独立 HTML 页面，内容围绕体育相关的笔记、资料整理与信息展示。每个页面均为自包含的 HTML 文件，可直接在浏览器中打开查看。

本项目不针对任何特定网站或平台，仅作为个人学习与资料整理的归档仓库。

## 目录说明

```
.
├── README.md            # 本文件
├── pages/               # 存放所有 HTML 页面
│   ├── index.html       # 页面导航索引（如有）
│   └── ...
└── assets/              # 可选的静态资源（图片、样式、脚本等）
    └── ...
```

- `pages/`：所有归档的 HTML 页面均放置于此目录下，每个页面独立且互不依赖。
- `assets/`：如有共享的 CSS、JavaScript 或图片资源，可放置于此目录。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖外部数据库或后端服务。
- 页面内容以体育相关笔记、数据整理、信息呈现为主。
- 页面可能包含本地样式与脚本，建议使用现代浏览器查看。
- 页面之间无强制跳转关系，可通过 `pages/index.html` 浏览全部页面列表（如有）。
